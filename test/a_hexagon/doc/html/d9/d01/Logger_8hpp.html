Detailed Description

High-performance async logging utility using spdlog.

Provides thread-safe, low-latency logging for real-time systems Async logging minimizes hot-path latency to ~20ns

Usage:

// Initialize once at startup (static function approach - c_hexagon style)
utils::Logger::init("a_hexagon");
// Or with file output
utils::Logger::initWithFile("a_hexagon", "logs/app.log");
// Use macros for logging
LOG_INFO("Application started");
LOG_DEBUG("Processing item {}", itemId);
LOG_ERROR("Failed to connect: {}", errorMessage);
// Shutdown at application exit
utils::Logger::shutdown();
