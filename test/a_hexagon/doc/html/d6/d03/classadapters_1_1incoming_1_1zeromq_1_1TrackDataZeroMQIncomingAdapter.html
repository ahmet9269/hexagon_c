Detailed Description

Receives TrackData via ZeroMQ DISH socket (UDP multicast)

Implements IAdapter interface for lifecycle management. Each instance runs in its own dedicated thread with real-time priority and CPU affinity support.

Invariant
When running_, receiveThread_ is active
socket_ is properly configured before start()
