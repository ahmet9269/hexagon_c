Detailed Description

Receives TrackData via IMessageSocket abstraction.

Implements IAdapter interface for lifecycle management. Each instance runs in its own dedicated thread with real-time priority and CPU affinity support. DIP compliant - socket is injected for testability.

Class Invariants:
When running_, receiveThread_ is active
