Detailed Description

Abstract interface for track data input port (primary port)

This interface defines the contract for receiving and processing TrackData from incoming adapters. Implements the "Driven" side of the hexagonal architecture.

Usage:

class TrackDataExtrapolator : public ITrackDataIncomingPort {
void processAndForwardTrackData(const TrackData& data) override {
// Process and extrapolate track data
}
};
