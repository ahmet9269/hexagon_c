Detailed Description

ZeroMQ UDP DISH Incoming Adapter for TrackData messages.

Implements the IAdapter interface for receiving TrackData messages via ZeroMQ DISH socket (UDP multicast). Runs in its own thread as part of Thread-per-Type architecture.
