Detailed Description

Mock implementation of IExtrapTrackDataOutgoingPort for unit testing.

Thread-safe recording of all sent ExtrapTrackData for verification

Features:

  • Records all sent ExtrapTrackData
  • Thread-safe operations
  • Call counting for verification
  • Configurable failure injection
  • Timing capture for latency analysis
