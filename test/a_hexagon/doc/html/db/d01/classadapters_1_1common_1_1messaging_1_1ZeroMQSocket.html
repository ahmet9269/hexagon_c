Detailed Description

ZeroMQ implementation of IMessageSocket interface.

Provides thread-safe wrapper around zmq::socket_t with proper resource management via RAII.

Supported socket types:

  • ZMQ_SUB: Subscribe socket (connects to PUB)
  • ZMQ_PUB: Publish socket (binds, others connect)
  • ZMQ_DISH: UDP multicast subscribe (Draft API)
  • ZMQ_RADIO: UDP multicast publish (Draft API)
Class Invariants:
context_ is always valid after construction
Class Invariants:
socket_ is null until connect() succeeds
