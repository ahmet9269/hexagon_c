Detailed Description

Manages all message pipelines with thread-per-type isolation.

Provides centralized lifecycle management for all adapters. Each registered pipeline operates independently.

Usage Example:

AdapterManager manager;
MessagePipeline trackPipeline("TrackData", trackInAdapter, trackOutAdapter);
manager.registerPipeline(std::move(trackPipeline));
if (manager.startAll()) {
// All pipelines running
}
manager.stopAll();
