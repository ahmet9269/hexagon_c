Detailed Description

Abstract interface for extrapolated track data output port (secondary port)

This interface defines the contract for sending extrapolated TrackData to outgoing adapters. Implements the "Driver" side of the hexagonal architecture.

Usage:

class ZeroMQOutgoingAdapter : public ExtrapTrackDataOutgoingPort {
void sendExtrapTrackData(const ExtrapTrackData& data) override {
// Serialize and send via ZeroMQ
}
};
